Q 1: How does Javascript work ?

Ans: Javascript ব্রাউজারে চলে V8 ইঞ্জিন দিয়ে । এর ভিতরে JTI(Just in time) compiler use করে কোড লাইন বাই লাইন interpreter করে, মানে এটা interpreter language। এর কিছু অংশ Synchronous, আবার কিছু অংশ Asynchronous ভাবে চলে। JavaScript নিজে single threaded। এটা multi threaded না।

V8 engine internal mechanism :
JS source code => Chrome V8 engine => Parsing => AST (Abstract Syntax Tree) Byte code => Machine code => Code runs on the CPU .

Q 2: What does Javascript Event Loop do ?

Ans: Event loop কে endless loop, main loop or main event loop,run loop বলা হয় । জাভাস্ক্রিপ্টের রানটাইম মডেল Event loop এর উপর ভিত্তি করে চলে। যেমন : code execute , collecting and processing events , executing queued sub-tasks ইত্যাদি কাজগুলো করে থাকে। Event loop সব সময় Call Stack এবং Callback Queue কে চেক করতে থাকে । যদি Call Stack খালি থাকে তাহলে event queue থেকে নতুন ফাংশন যেটা প্রথমে আছে সেটা যোগ করে। যদি Call Stack খালি না থাকে তাহলে current function টাকেই call করে। তারপর ব্রাউজার কনসোল এবং Call Stack খালি করে। তারপর সে ঘুমায় মানে এটি কাজের জন্য অপেক্ষা করতে থাকে
